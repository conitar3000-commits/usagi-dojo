# うさぎ道場

うさぎ定石からローズ系の分岐を、4手ずつ反復して覚えるブラウザゲームです。

## 起動

依存パッケージはありません。`index.html` をブラウザで開くか、ローカルサーバーを起動します。

```sh
python3 -m http.server 4173
```

その後 `http://localhost:4173` を開いてください。

## iPhoneに追加

公開URLをiPhoneのSafariで開き、共有ボタンから「ホーム画面に追加」を選びます。追加後はホーム画面の「うさぎ道場」アイコンから、独立したアプリ表示で起動できます。一度読み込めばオフラインでも利用できます。

## 収録ドリル

- ローズ基本形 (S)
- ローズフラット (F)
- 三村流
- 大和久流
- 井上流
- 手塚システム

各ドリルは選んだ定石の正しい局面から開始し、黒白交互に最善手を4手回答します。誤答するとその場でゲームオーバーになります。同評価の最善手が複数ある場合は、どれを選んでも正解です。

局面と評価値は Othello! JAPAN の評価エンジン（Edax）で確認しています。クリア回数はブラウザの `localStorage` に保存されます。
